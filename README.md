# veda_qt_todo

## 완성한것
- todo 입력, 출력
- todo 수정
- todo 완료 체크
- 유저 로그인, 로그아웃
- 저장(이진파일)

## 코드상 있지만 아직 하지 못한 것
- 마감일 이후 todo 내용의 글자색을 빨간색으로 표시
- text로 출력
